## 🔧 如需进一步调整

### 如果觉得文字还不够大
编辑 `MainWindow.cpp` 的 `setupStyles()` 函数：

```cpp
QLabel {
    font-size: 18px;  // 再增大 2px
}

QLineEdit {
    font-size: 20px;  // 再增大 2px
}

QPushButton {
    font-size: 20px;  // 再增大 2px
}
```

### 如果想更改按钮颜色
编辑 `MainWindow.cpp` 的 `setupStyles()` 函数：

```cpp
QPushButton#btnStart {
    background-color: #28a745;  // 更改为其他绿色
}

QPushButton#btnStop {
    background-color: #dc3545;  // 更改为其他红色
}
```
